# PaperPlus for Paper/Paper Lite
PaperPlus는 리디북스 페이퍼/페이퍼 라이트의 페이지 넘김버튼을 다른 이북/전자도서관앱에서도 사용할 수 있게 지원해줍니다.

- 이북모드에서만 사용가능합니다. (PDF모드는 지원하지 않음)
- 페이지 넘김 이외에 다른 기능은 지원하지 않습니다. 

### 지원하는 앱
##### 전자도서관
- 교보문고
- 이피루스
- 예스24
- 북큐브
- 서울도서관 (북큐브, YES24)

##### 이북
- 교보문고
- 예스24
- 카카오페이지
